# Admin Panel 
## Quickstart (local)
1. Start MySQL and create database `admin_panel`.
2. Backend:
   - cd backend
   - copy `.env.example` to `.env` and set values
   - npm install
   - npm run dev
3. Frontend:
   - cd frontend
   - npm install
   - npm run dev
Backend listens on port 3001 (configurable via .env). Frontend Vite dev server runs on 5173.
